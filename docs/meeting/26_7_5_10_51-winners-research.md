## Key Finding: Winners of the gstack × gbrain Hackathon #1 (5/22, 200+ participants, 87 submissions)

| Rank | Project | What They Built |
|---|---|---|
| 1st | **Build your Phone** (Sina Matian) | A framework that gives Claude iPhone development skills, physically operating a real iPhone to auto-fix bugs. Automatically maps the entire SwiftUI source, exposes state in real time via a CoreDevice USB tunnel, and automates `find→fix→verify` |
| 2nd | **LearningGraph** | An adaptive learning tool using GBrain as a memory layer, dynamically adjusting lessons based on proficiency and visualizing knowledge as a graph |
| 3rd | **GBody** | Self-optimization in physical space using a robotic arm + MCP + Claude + GStack. Demonstrates agents advancing into the physical domain |
| 4th | **Hindsight** | A self-reflection layer for GBrain |

Special Awards: Conamur (retail AI voice agent / persistent customer memory), Cortex (Duolingo-style app for medical students), OrderUp (restaurant OS), Docket (PACER litigation records × GBrain litigation strategy engine), and others.

## Trends in What Wins (Judging Perspective)

1. **Things that "really work" on real hardware/physically** score highest (both 1st and 3rd place involved hardware integration, with demos that conveyed value instantly)
2. **The judging criteria are "the same as a YC application."** Judges have firm opinions and evaluate based on whether the intent can be understood instantly
3. Common behavior among winners: stress-test the idea with peers and judges → build aggressively in a clear direction → build in line with the prompt (RFS)
4. Projects that deeply integrated sponsor technology (GBrain = memory layer / GStack = skills / The Hog / ZeroEntropy / Lightsprint) won each award (about 60% of entries were in the GBrain track)
